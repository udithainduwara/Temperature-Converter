# Temperature-Converter
BSc (Hons) Data Science past papers and solutions – Sabaragamuwa University

# Project Name
Temperature Convertor

## Description
Simple HTML, CSS, and JavaScript Temperature Converter (Celsius ↔ Fahrenheit)

## Features / Contents
- Convert Celsius to Fahrenheit and vice versa
- User-friendly colors and input box and dropdown for conversion selection
- Styled button with hover effect 

## How to Use / Run
1. Enter your temperature in the input box
2. Select what you want to convert (Celsius → Fahrenheit or Fahrenheit → Celsius)
3. Click the **Convert** button
4. See the converted temperature displayed below the button
